# Miheret Naturals — Demo Frontend

Quick Vite + React demo for the Miheret Naturals storefront and admin UI.

Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

Notes

- This project uses Tailwind CSS. If you don't want Tailwind, remove the PostCSS and Tailwind files and the `@tailwind` directives in `src/index.css`.
- Use username `admin` to view the admin dashboard or any other value to log in as a demo customer.
